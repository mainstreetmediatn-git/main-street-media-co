# Framer Reference Notes

Reference URL: https://shining-osprey-291735.framer.app/

## Page Structure

- Sticky header with MS mark, Main Street Media Co. brand text, and a free audit CTA.
- Large hero section focused on website authority and local service business visibility.
- Hero includes a right-side "Website Authority Snapshot" dashboard card.
- Trust/proof strip appears near the hero with Local SEO, Trust Signals, and Call Conversion.
- Local visibility gap section explains why a pretty website is not enough.
- Web Authority Framework section uses numbered steps: Findability, Trust, Clarity, Conversion, Expansion.
- "What we fix first" section lists tactical website and local SEO issues.
- Industries section uses compact industry chips.
- Before/after section contrasts weak current state against authority rebuild outcomes.
- Final free visibility audit CTA section repeats the audit offer and no-guarantee positioning.
- Footer-style closing CTA: "Become the Business Customers Find First."

## Hero Layout

- Two-column desktop hero.
- Left column: brand promise, large headline, concise paragraph, primary and secondary CTAs, and small proof chips.
- Right column: dark glassmorphism dashboard with a score, audit label, diagnostic categories, and short bullet list.
- Headline from reference: "Build a Website Google Trusts and Customers Choose."
- Hero message is direct, conversion-focused, and built around website authority rather than generic agency services.

## Visual Style

- Premium dark SaaS/local agency look.
- Dark green-black background with soft gold and green highlights.
- Glassy cards with subtle borders, rounded corners, and layered depth.
- High-contrast warm text, restrained gold accents, and green support color.
- Clean grid sections with confident spacing.
- Section headings are large and direct.

## Colors

- Background: near-black with green undertone.
- Primary accent: warm gold.
- Secondary accent: muted green/teal.
- Text: warm off-white.
- Muted copy: translucent warm off-white.
- Cards: semi-transparent dark panels with light borders.

## Typography Feel

- Modern sans-serif.
- Large, tight hero typography.
- Bold section headings.
- Small uppercase eyebrow labels for section context.
- Body copy is plain-English and practical.

## CTA Placement

- Primary CTA in header.
- Primary and secondary CTA in hero.
- Audit CTA repeated in final conversion section.
- CTA language centers on requesting a free visibility audit.

## Mobile/Responsive Behavior

- The public text output does not expose viewport screenshots, but the structure suggests stacked mobile sections.
- Important responsive needs for the coded version: collapse hero to one column, hide or simplify nav links, keep CTAs full-width where needed, stack card grids, and keep dashboard/card text readable.

## Design Strengths to Preserve

- Clear authority positioning around ranking, trust, and conversion.
- Strong hero dashboard concept.
- Practical diagnostic framing instead of vague marketing claims.
- Numbered framework communicates process clearly.
- Industry chips make target market obvious.
- No fake testimonials or ranking guarantees.

## Weaknesses to Improve in Code

- Avoid repeated headline/body text that appears in the Framer text output.
- Use cleaner semantic React components and reusable data structures.
- Add a real WebAuthority article hub route rather than a single-page-only experience.
- Ensure all links resolve in the coded site.
- Improve content scalability for future long-form articles.
- Keep responsive behavior explicit in CSS rather than relying on Framer layout output.

## Scope Confirmation

- The reference was used only as a visual and structural guide.
- No Framer export code was copied.
- No Framer dependency was added.
- No files from SentinalOS, Agent Horde, shannon-uncontained, or any other folder outside this repository were used.
